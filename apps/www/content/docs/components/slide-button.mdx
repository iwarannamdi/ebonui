## Props

| Prop              | Type              | Default            |
| ----------------- | ----------------- | ------------------ |
| `delay?`          | `time`            | `#ffffff`          |
| `fillHeight?`     | `percentage`      | `0.05em`           |
| `hoverScale?`     | `number`          | `100px`            |
| `tapScale?`       | `number`          | `3s`               |
| `...props?`       | `HTMLMotionProps<"button">`|           |
