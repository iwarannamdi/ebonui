## Props

| Prop        | Type     | Default | Description                                                |
| ----------- | -------- | ------- | ---------------------------------------------------------- |
| `className` | `String` | " "     | Additional classes to be added to the component            |
| `duration`  | `Number` | `200`   | Duration of the theme transition animation in milliseconds |
