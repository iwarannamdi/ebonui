## Legacy Version (Tailwind v3)

For older Ebon UI components using Tailwind CSS v3, visit: [v3.ebonui.com](https://v3.ebonui.com)

## Upgrading

To upgrade from legacy to current version, follow the migration guide: [https://ui.shadcn.com/docs/tailwind-v4](https://ui.shadcn.com/docs/tailwind-v4)
